Mã hóa dữ liệu từ vựng — chỉ chạy được trên domain của bạn

1. Điền đúng domain website của bạn
2. Chọn file tuvung.json gốc
3. Bấm "Tạo file mã hóa" → tải về
4. Đưa file tuvung.enc.json lên GitHub (vào thư mục /data/)
